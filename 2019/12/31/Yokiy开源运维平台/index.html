Yokiy 运维自动化平台项目

  • 项目地址：https://gitee.com/jdytrip/jdy_devops.git
  • DEMO: http://demo.yokiy.com/ 用户名 admin 密码 admin123

项目简介

  • 这是一个web自动化平台，实现一键部署。
  • 基于Python 3 和 Django2.0
  • Bootstrap ampleadmin 模板
  • Docker kubernetes
  • Mysql 5.6 & Redis 4.0

主要功能

  1. 用户权限管理
  2. CI/CD 持续部署，持续构建
  3. MySql执行管理
  4. 应用日志管理
  5. 应用监控
  6. API监控

依赖

  • django==2.0
  • requests==2.18.4
  • mysqlclient==1.3.12
  • PyMySQL==0.8.0
  • pycurl==7.43.0.1
  • APScheduler==3.0.3
  • SQLAlchemy==1.1.7
  • celery==4.1.0
  • django-celery-results==1.0.1
  • django-redis==4.8.0
